- 🚀 빠른 성능
- 💎 세련된 디자인
- 📱 반응형 디자인
- ✨ 인터랙티브한 애니메이션

## GitHub Pages로 배포하기

1. GitHub 저장소에서 Settings → Pages 이동
2. Source에서 Branch를 `main`으로 선택
3. Folder를 `/ (root)`로 선택
4. Save 클릭

몇 분 후 다음 URL로 접속할 수 있습니다:
`https://YOUR_USERNAME.github.io/test/`

## 로컬에서 실행하기

브라우저에서 `index.html` 파일을 열거나, 로컬 서버를 실행하세요.
