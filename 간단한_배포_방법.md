# 간단한 배포 방법 (가장 쉬움!)

## 방법 1: Netlify Drop (가장 쉬움! 드래그 앤 드롭만)

1. https://app.netlify.com/drop 에 접속
2. 이 폴더의 `index.html` 파일을 브라우저로 드래그 앤 드롭
3. 끝! 바로 URL이 생성됩니다!

**장점**: 
- 회원가입 없이 사용 가능
- 1분이면 끝
- 자동으로 HTTPS 제공

## 방법 2: GitHub Pages (무료, 안정적)

### GitHub Desktop 사용 (추천!)

1. **GitHub Desktop 다운로드**: https://desktop.github.com/
2. GitHub Desktop 설치 후 실행
3. File → Add Local Repository
4. 이 폴더 선택 (`d:\YJH\개인파일\00 cursor`)
5. Publish repository 클릭
6. GitHub.com에 로그인하고 저장소 이름 입력
7. GitHub 저장소 페이지에서 Settings → Pages → Source를 `main`으로 설정

### VS Code Git 사용

VS Code에서:
1. 왼쪽 사이드바에서 Source Control 아이콘 클릭 (또는 `Ctrl + Shift + G`)
2. "Initialize Repository" 클릭
3. "Changes" 옆의 `+` 버튼으로 파일 추가
4. 커밋 메시지 입력 후 "Commit" 버튼 클릭
5. "Publish Branch" 버튼 클릭
